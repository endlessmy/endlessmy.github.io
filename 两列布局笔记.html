              # 绝对定位布局

              绝对定位布局时如果不给父元素设置高度，会导致底绝对定位的部分覆盖在底下页面的上面。这是因为父元素高度没有被撑开，虽然子元素设置了高度，但对父元素没用。解决方法是给父元素设置高度

              .content2 {
    position: relative;
}
.content2 .slider {
    position: absolute;
    left: 0;
    width: 30%;
    height: 100px;
    background: yellow;
}
.content2 .main {
    position: absolute;
    right: 0;
    width: 70%;
    height: 100px;
    background: blue;
}

              # flex 布局

              flex 布局的里的元素是并排排列的。可以理解为行内元素。

              .content3 {
    display: flex;
}
.content3 .main {
    flex-basis: 70%;
    height: 100px;
}
.content3 .slider {
    flex-basis: 30%;
    height: 100px;
}

              # inline-block 布局

              inline-block 布局有个问题是它元素之间是有间隙的，当一个为 30%，另一个为 70%时，一排是放不下的。解决方法是给父元素设置letter-spacing:-10px，或者给父元素设置font-size:0，子元素恢复font-size。但使用letter-spacing会导致元素里面的字体会黏到一起。所以最好还是使用font-size解决。

              # 代码如下

              <!DOCTYPE html>
<html lang="zh">
    <head>
        <meta charset="UTF-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <meta http-equiv="X-UA-Compatible" content="ie=edge" />
        <title>Document</title>
    </head>
    <style>
        .main {
            background: red;
        }
        .slider {
            background: green;
        }

        /*方案1*/
        .content1 .main {
            width: 70%;
            height: 100px;
            float: right;
        }
        .content1 .slider {
            width: 30%;
            height: 100px;
            float: left;
        }

        /*方案2*/
        .content2 {
            position: relative;
            height: 100px;
            /* overflow: hidden; */
        }
        .content2 .main {
            position: absolute;
            right: 0;
            width: 70%;
            height: 100px;
        }
        .content2 .slider {
            position: absolute;
            left: 0%;
            width: 30%;
            height: 100px;
        }

        /*方案3*/
        .content3 {
            display: flex;
        }
        .content3 .main {
            flex-basis: 70%;
            height: 100px;
        }
        .content3 .slider {
            flex-basis: 30%;
            height: 100px;
        }

        /*方案4*/
        .content4 {
            /* letter-spacing: -10px; */
            font-size: 0;
        }
        .content4 .slider {
            width: 30%;
            height: 100px;
            display: inline-block;
            font-size: 16px;
        }
        .content4 .main {
            width: 70%;
            height: 100px;
            display: inline-block;
            font-size: 16px;
        }
    </style>

    <body>
        1、方案1使用的float布局
        <div class="content1">
            <div class="main">main</div>
            <div class="slider">slider</div>
        </div>

        2、方案2使用absolute布局
        <div class="content2">
            <div class="slider">slider</div>
            <div class="main">main</div>
        </div>

        3、方案3使用flex布局
        <div class="content3">
            <div class="slider">slider</div>
            <div class="main">main</div>
        </div>

        4、方案4使用inline-block布局
        <div class="content4">
            <div class="slider">slider</div>
            <div class="main">main</div>
        </div>
    </body>
</html>
