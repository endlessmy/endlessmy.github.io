  • 双飞翼布局笔记

    • 双飞翼布局要求
      • 双飞翼布局的实现
        • 代码如下

        双飞翼的布局和圣杯布局的效果相同，都是两侧固定，中间自适应。不同之处在于解决中间部分被遮挡的问题时，采取的办法不一样。双飞翼布局是通过给中间元素 div 的外面再包一个 div。再给这个新的 div 设置 margin-left 和 margin-right。

        # 双飞翼布局要求

        1、header 和 footer 各自占领屏幕所有宽度，高度固定。
        2、中间的container是一个三栏布局。
        3、三栏布局两侧固定不变，中间自动填满整个区域。
        4、中间部分的高度是三栏中最高区域的高度。

        # 双飞翼布局的实现

        1、left、center和right 三种都设置浮动。
        2、设置 center 宽度为 100%。
        3、设置负编剧，left负编剧设为100%，right负编剧设为自身宽度。
        4、设置 content 的 margin 值为左右两个侧栏留出空间，margin 值大小为 left 和 right 宽度。

        # 代码如下

        <!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <meta http-equiv="X-UA-Compatible" content="ie=edge" />
        <title>双飞翼布局</title>
    </head>
    <style>
        .header {
            width: 100%;
            height: 50px;
            background: #cccccc;
            text-align: center;
        }
        .container {
        }
        .container .left {
            float: left;
            width: 50px;
            height: 500px;
            background: pink;
            margin-left: -100%;
        }
        .container .center {
            float: left;
            width: 100%;
            height: 500px;
            background: blue;
        }
        .container .right {
            float: left;
            width: 150px;
            height: 500px;
            margin-left: -150px;
            background: red;
        }
        .footer {
            width: 100%;
            height: 50px;
            background: #cccccc;
            text-align: center;
        }
        .container .content {
            margin: 0 150px 0 200px;
        }
    </style>
    <body>
        <div class="header">header部分</div>
        <div class="container">
            <div class="center"><div class="content"></div></div>
            <div class="left"></div>
            <div class="right"></div>
        </div>
        <div class="footer">footer部分</div>
    </body>
</html>
