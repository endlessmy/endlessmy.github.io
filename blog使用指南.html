    # npm 命令介绍

    # 博客开发环境

    npm start

    # 静态页面开发环境

    npm run static

    # 打包（一般不用）

    npm run build

    # 生成文章

    npm run post <name> [template]

    • name: 文章标题
    • template: 模版，默认是default.md 可以在 docs/.vuepress/cli/post/templates 中自定义，然后在docs/.vuepress/cli/post/index.js中引用

    # 静态文件

    在docs/.vuepress/public/文件下都是静态目录，会被解析到根目录 比如 docs/.vuepress/public/image解析成/image。可以新建static目录来存在静态的html文件

    可以通过 npm run static 来启动静态文件开发的服务
