        # flex 布局简介

        任何元素都可以设置 flex 布局。

        .box {
    display: flex;
}

        行内元素可以设置display: inline-flex flex 中父元素称作容器，子元素称为项目。

        # 容器的属性

        容器默认有两根轴，主轴和交叉轴。容器有以下属性： 1、flex-direction属性决定主轴的方向。 2、flex-wrap属性决定容器中项目一行排不下时，如何换行。 3、flex-flow属性是上面两种的简写，默认为row nowrap。 4、justify-content属性定义项目在主轴上的对齐方式。 5、align-item属性定义项目在交叉轴上如何对齐。 6、align-content属性定义多根轴线的对齐方式。如果只有一根轴线，该属性不起作用。

        # 项目的属性

        1、order 属性定义项目的排列顺序。数值越小，排列越靠前，默认为 0。 2、flex-grow 属性定义项目的放大比例。默认为 0，即存在剩余空间，也不放大。 3、flex-shrink 属性定义项目的缩小比例。默认为 1，即空间不足，该项目将缩小。 4、flex-basis 属性定义项目占据的主轴空间，也就是 width 5、flex 属性前面三个的简写，默认为 0 1 auto。我们在写自适应时，项目一般设置 flex:1。 6、align-self 属性单个项目与其他项目的不一样的对齐方式。

        # 实例

        HTM 模板如下：

        <div class="box">
    <span class="item"></span>
</div>

        div {
    height: 500px;
    border: 1px solid red;
}

        # 单项目垂直居中对齐

        .box {
    display: flex;
    justify-content: center;
    align-items: center;
}

        # 双项目

        <div class="box">
    <span class="item"></span>
    <span class="item"></span>
</div>

        1、双项目主轴两端对齐

        .box {
    display: flex;
    justify-content: space-between;
}

        2、双项目交叉轴两端对齐

        .box3 {
    display: flex;
    flex-direction: column;
    justify-content: space-between;
}
