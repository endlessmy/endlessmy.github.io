  • 两列布局之扩展

    • float 布局
      • 绝对定位布局
        • flex 布局
          • overflow 布局
            • 代码如下

            之前实现的两列布局是左侧跟右侧都是固定的，这次实现左侧固定，右侧自适应的布局。

            # float 布局

            float 和 margin-left 一起搭配使用

            # 绝对定位布局

            .content2 {
    position: relative;
    height: 200px;
}
.content2 .slider {
    width: 200px;
    height: 200px;
    left: 0;
    position: absolute;
}
.content2 .main {
    height: 200px;
    left: 200px;
    right: 0;
    position: absolute;
}

            # flex 布局

            .content3 {
    display: flex;
}
.content3 .slider {
    flex: 0 0 200px;
    height: 200px;
}
.content3 .main {
    flex: 1;
    height: 200px;
}

            flex 属性是flex-grow、flex-shrink和flex-basis的简写，默认为0 1 auto，后两个属性可选。 其中flex-grow表示元素的放大比例，默认为 0，如果存在剩余空间，也不放大。 flex-shrink表示元素的缩小比例，默认为 1，即如果空间缩小，元素将缩小。 flex-basis表示元素占据空间的大小，相当于 width。

            # overflow 布局

            .content4 {
}
.content4 .slider {
    width: 200px;
    height: 200px;
    float: left;
}
.content4 .main {
    overflow: hidden;
    height: 200px;
}
